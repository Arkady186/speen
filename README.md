Speen TG Mini App (каркас)

Запуск локально:
1. npm i
2. npm run dev

Фронтенд: http://localhost:5173
Бэкенд: http://localhost:8080

Состав:
- apps/web — Vite + React + TypeScript, TonConnect UI
- apps/server — Express API (заглушки), хранение в apps/server/data/db.json
